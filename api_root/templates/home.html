Bem-vindo(a) à API ClientConnect

API do sistema de gerenciamento de contatos desenvolvido em Django Rest Framework. O sistema permite a criação, edição, exclusão e listagem de contatos, com a funcionalidade de alta performance e consistência transacional. A API foi projetada para integrar-se com um banco de dados PostgreSQL e utiliza Redis para cache.
